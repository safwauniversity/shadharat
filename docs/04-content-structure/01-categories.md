# الأقسام (Categories)

صفحة خاصة تجمع كل الأقسام الرئيسية:
*   بودكاست
*   ديني
*   ثقافي
*   ترفيهي
*   أطفال
*   قصص
*   برامج حصرية
*   لقاءات
*   فتاوى

ويمكن إضافة أقسام جديدة بسهولة من لوحة التحكم.

## قسم "اكتشف جديد"
*   يظهر محتوى من أقسام مختلفة بطريقة ديناميكية.
*   مثالي لمحتوى ثقافي أو ترفيهي قصير خارج المألوف.

## قسم المحتوى المميز / الحصري
*   مكان لعرض حلقات أو برامج حصرية للمشتركين فقط.
*   يمكن تسميته: "حصريات شذرات" أو "المميز اليوم".
