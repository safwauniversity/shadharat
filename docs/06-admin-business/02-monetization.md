# نموذج العمل والدفع

## الاشتراكات والدعم
*   **Micro-donations**: زر تبرع للمحتوى أو لدعم الشيخ/السلسلة.
*   **Daily Shathra Push**: شذرة صوتية أو نصية يومية بالاشتراك.
*   **خطط اشتراك**: Free (ad-light) وPremium (بدون إعلانات + تنزيل). (يمكن تفعيل لاحقًا).

## الدفع
*   دعم البطاقات/موفري الدفع المحليين في السوق المستهدف.
