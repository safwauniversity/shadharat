# واجهة المستخدم الرئيسية (App Home UX)

عند فتح التطبيق لأول مرة، تظهر للمستخدم واجهة منظمة وسهلة التصفح تتكون من:

## 1) قسم "الأعلى مشاهدة"
أول سطر في الشاشة يعرض الفيديوهات والحلقات الأكثر مشاهدة خلال الأسبوع/الشهر، لإظهار المحتوى الأقوى والأكثر طلبًا.

## 2) قسم "المقترحات"
شريط يظهر محتوى مقترح بناءً على:
*   تصنيفات عامة
*   المحتوى الشائع
*   **ملاحظة**: بدون توصيات عمرية أو شخصية (كما طلبت).
