# البحث والاستكشاف

## محرك بحث جزئي متقدم
*   بحث سريع يعتمد على الكلمات المفتاحية.
*   دعم البحث الصوتي.
*   ترشيح المحتوى حسب الفئة، النوع، المدة، والمشايخ.

## ميزة البحث داخل الفيديوهات
*   البحث في العناوين، الوصف، وحتى تفريغ النصوص الصوتية (Transcripts).
*   المستخدم يقدر يكتب كلمات مفتاحية ويظهر له اللحظة الزمنية في الفيديو.
