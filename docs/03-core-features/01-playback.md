# خصائص التشغيل

## مشغل الفيديو
*   جودة قابلة للتغيير.
*   تحكم بسرعة التشغيل.
*   Captions (ترجمة/نصوص).
*   حفظ آخر نقطة مشاهدة (resume).

## التشغيل في الخلفية
*   دعم تشغيل صوتي بعد إغلاق التطبيق (Android foreground service، iOS background audio).
*   دعم PIP (صورة داخل صورة) على الأجهزة المدعومة.
