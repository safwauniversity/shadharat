# قوائم التشغيل الذكية

## قوائم التشغيل (Playlists)
*   يستطيع المستخدم إنشاء قوائم تشغيل خاصة.
*   قوائم تلقائية مثل: المحتوى القصير – القرآن – الدروس اليومية – الأطفال – العلوم الشرعية.

## ميزة قائمة التشغيل الشخصية
*   المستخدم يقدر ينشئ Playlist خاص به، يمزج بين الشورتس، البودكاست، والفيديوهات الطويلة.
*   مفيد للمتابعة اليومية أو الدراسة.
